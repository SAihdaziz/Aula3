Introdução ao HTML

Introdução ao HTML:Html significa HyperText Markup Language, que em português pode ser entendido como Linguagem de Marcação de Hipertexto. Ele é a base de qualquer página da internet e serve para estruturar o conteúdo que será exibido no navegador.

INSTRUÇÕES: Quando você acessa um site, o navegador lê o código HTML e entende:

  1. onde está o título
  2. onde estão os textos
  3. imagens
  4. links
  5. formulários
  6. seções da página

O HTML não é uma linguagem de programação, porque não possui lógica, cálculos ou decisões. Ele é uma linguagem de marcação, usada apenas para organizar e descrever o conteúdo.

Estrutura básica de uma página HTML

Estrutura básica de uma página HTMLToda página HTML possui uma estrutura padrão que informa ao navegador como o conteúdo deve ser interpretado e exibido. Essa estrutura é obrigatória para que a página funcione corretamente.

A estrutura básica de uma página HTML é composta por:

  • Doctype: define o tipo de documento HTML
  • HTML: elemento raiz da página
  • Head: contém metadados e informações sobre a página
  • Body: contém todo o conteúdo visível da página
  • leitores de tela
  • modo de acessibilidade
  • mecanismos de busca

Declaração do tipo de documento (DOCTYPE)A primeira linha de um arquivo HTML informa ao navegador qual versão do HTML está sendo usada. Hoje, utiliza-se o HTML5. Essa declaração garante compatibilidade com navegadores modernos e evita erros de interpretação. Todo o conteúdo da página fica dentro de um elemento principal. Elementos HTML

Elementos HTML — Explicação Detalhada

Elementos HTML: Um elemento HTML é formado por uma tag de abertura, conteúdo e uma tag de fechamento. Por exemplo, <h1>Título</h1>.

O que é uma TAG HTML: Uma tag HTML é uma instrução que define como o conteúdo deve ser exibido no navegador. Ela é composta por um nome entre sinais de menor e maior, como <h1> ou <p>. As tags podem ser abertas e fechadas ou auto-fechadas.

  • Tipos de Tags HTML:
  • p → parágrafo
  • h1 → título principal
  • a → link
  • img → imagem
  • div → contêiner genérico

Tipos de Elementos HTML

Tipos de Elementos HTML: Os elementos HTML podem ser classificados em três tipos principais:

  1. Inline: elementos que não quebram a linha, como <span>, <a>, e <img>.
  2. Block: elementos que ocupam toda a largura disponível, como <div>, <p>, e <h1>.
  3. Self-closing: elementos que não precisam de uma tag de fechamento, como <br />, <hr />, e <img src="imagem.jpg" />.
